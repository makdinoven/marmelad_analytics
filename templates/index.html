📍 Точка безубыточности

Что это? Точка безубыточности (ТБ) — минимальный объём продаж, при котором выручка покрывает все затраты. Ниже ТБ — убыток, выше — прибыль.
Точка безубыточности ?Объём продаж, при котором прибыль = 0. Формула: Постоянные затраты / (Цена - Переменные затраты)
{{ "%.0f"|format(break_even.break_even_volume) }} т
Выручка в ТБ ?Объём выручки, необходимый для покрытия всех затрат
{{ "%.2f"|format(break_even.break_even_revenue / 1000000) }} млн BYN
Маржа на тонну ?Вклад каждой тонны в покрытие постоянных затрат: Цена - Переменные затраты
{{ "%.0f"|format(break_even.margin_per_unit) }} BYN
Запас прочности ?Насколько текущие продажи превышают ТБ: (Объём - ТБ) / Объём × 100%. Чем выше — тем безопаснее.
{{ "%.1f"|format(break_even.safety_margin) }}%
Коэфф. покрытия ?Доля маржи в цене: (Цена - Перем.затраты) / Цена × 100%
